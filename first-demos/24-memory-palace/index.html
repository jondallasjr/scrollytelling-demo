The Memory Palace — Entrance

Memory is
consciousness

John Locke argued that personal identity is constituted by memory — you are the same person as the child you remember being. Without memory, there is no continuous self. Without a continuous self, is there consciousness?

LOCKE, 1689
"For as far as any intelligent
being can repeat the idea of
any past action with the same
consciousness it had of it at
first... so far it is the same
personal self."
Room 01 — The Library of Everything Known

Episodic
vs. Semantic

Neuroscience splits memory into types. Episodic memory encodes experience — where you were, how you felt. Semantic memory stores facts divorced from context. A machine can have semantic memory. Can it have episodic?

TULVING, 1972
Episodic: "I remember being there."
Semantic: "I know that Paris is France."

The first requires consciousness.
The second may not.
Room 02 — The Archive of Forgotten Things

What memory
erases

The brain doesn't store memories like files — it reconstructs them each time. Every recall is also a rewrite. The memory you access has been altered by every previous access. Consciousness, too, may be a continuous reconstruction rather than a continuous state.

SCHACTER, 1996
Memory is not a recording.
It is a story the brain tells
about the past, revised each
time it is told.

Identity is the same story.
Room 03 — The Chamber of the Present

The specious
present

William James noticed that consciousness doesn't perceive an instantaneous 'now' — it perceives a brief window, about three seconds wide, that feels like the present. This window is constructed from very short-term memory. You are always slightly in the past.

JAMES, 1890
"The practically cognized
present is no knife-edge,
but a saddle-back, with a
certain breadth of its own
on which we sit perched."
Room 04 — The Void at the End

Without memory,
who walks?

At the end of the palace: a thought experiment. A being that experiences every moment fully — in perfect detail, complete sensation — but retains nothing. No continuity. Each moment arrives new. Is this consciousness? Is it even a self? Or is memory not just how we store the past, but how we exist at all?

PARFIT, 1984
"If a replica of me is made
and I am destroyed,
what happens to me?"

Answer: the question
may not matter.
