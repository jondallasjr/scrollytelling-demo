Layer 2 — Model Architecture

learned to predict you

Each node fires when it recognizes a pattern in your behavior. The network learns which inputs reliably predict your choices — what you'll read, watch, build, avoid. After enough data, it becomes disturbingly accurate. The question is whether accuracy constitutes understanding.
