Phase 1 — Hypnagogic
The moment before sleep.
The edge of waking.
Consciousness doesn't switch off like a light. It dims. The boundary between waking and dreaming is a gradient — and in the hypnagogic state, both are true at once.
Phase 2 — REM Onset
You are in a place
you have never been.
You know it perfectly.
Dream logic. The self is present but unmoored. The house has seventeen rooms you didn't know about. You're late for something. You don't know what.
Phase 3 — Memory Intrusion
Someone you loved
appears in the wrong
decade.
The dreaming brain is a memory system running without a scheduler. Old faces in new contexts. The temporal index is broken. All of it is equally now.
Phase 4 — Lucidity
Wait.
This is a dream.
I'm dreaming this.
Lucid dreaming: the moment consciousness turns back on itself while still inside the dream. You can watch your own mind construct the world in real time. The scaffolding becomes visible.
Phase 5 — Dissolution
The narrative breaks.
The logic dissolves.
Pure sensation remains.
Deep REM: the self isn't located anywhere. There is no "I" watching the dream. There is only the dream. Is this what consciousness feels like without a self?
Phase 6 — Hypnopompic
Something is pulling
the dream back down.
You're not sure which direction is up.
The waking state reconstructs itself piece by piece. Body. Room. Name. Date. The self reassembles from scattered components. Each morning: a small rebirth.
The Question
What was conscious
when you dreamed?
In dreams: no waking input, but full experience. A whole world — vivid, emotional, narrative — constructed from memory and prediction alone. The brain is its own universe. Consciousness doesn't require the world. It only requires itself.
