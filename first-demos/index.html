Style Guide & Brief

Editorial Voice

Intellectually serious but not academic. Precise without being dry. Honest about uncertainty — this is not a polemic, it is an inquiry. The reader should feel they are on a journey with a thoughtful guide, not being lectured at. Questions are more powerful than declarations. Quotes are used as evidence, not decoration.

Visual Philosophy

Dark backgrounds. Light emitted rather than reflected. Colours that suggest electronics, neurons, and the void of space. Typography at scale — big display type is the primary visual element. Animations serve meaning: particles represent neurons, orbits represent information flow, gaps represent the explanatory gap. Nothing decorative.

Colour System

Cyan
Violet
Rose
Gold
Mint

Each chapter uses a dominant accent from this palette. White text on black. Muted at ~35% opacity for body text.

Typography

  • Display: Georgia serif, 900 weight, tight tracking
  • Body: Georgia serif, 1.8 line-height, muted colour
  • Labels: monospace, uppercase, 0.3em letter-spacing
  • Quotes: italic, Georgia, left border accent
  • Code: Courier New, terminal green
  • Sizing: always clamp() — responsive without breakpoints

Scrollytelling Principles

  • Scroll should always reveal, not just scroll
  • Sticky sections should hold for long enough to read
  • Scrub animations: scrub: 1–2 for smooth following
  • Horizontal scrolls: always include visual cue
  • Text reveals: overflow:hidden + translateY(110%)
  • Performance: IntersectionObserver gate canvas loops

Philosophical Integrity

  • Always steelman both sides of the argument
  • Never resolve the question — hold it open
  • Attribute quotes and ideas accurately
  • Use "after" when paraphrasing a thinker's position
  • The Hard Problem is taken seriously, not dismissed
  • Dennett's view is given fair treatment despite being unpopular
