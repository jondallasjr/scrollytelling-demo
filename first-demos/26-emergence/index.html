Project 26 — The Emergence Engine

Consciousness
from nothing but rules

Simple rules. Complex behavior. A single cell following three conditions generates infinite patterns. An ant with two states draws a highway after 10,000 steps. Where does the complexity come from, if not from the rules themselves?

System 01 — Conway's Game of Life

Four rules,
infinite complexity

Born, live, die, survive. Four rules applied simultaneously to every cell. From these rules: gliders, spaceships, oscillators, patterns that compute. The emergence of structure from pure logic. No designer. No goal. Just rules iterating.

System 02 — Langton's Ant

10,000 steps
to order

Two states. One rule.

An ant on a grid: if white, turn right, flip to black, move. If black, turn left, flip to white, move. For ~10,000 steps: apparent chaos. Then suddenly — without any change to the rules — the ant starts building a diagonal highway that extends forever. The order was always implicit in the chaos.

System 03 — Wolfram's Rules

256 universes,
one rule

Every 1D cellular automaton can be encoded as a number 0–255. Rule 30 is chaotic. Rule 90 is fractal. Rule 110 is Turing complete. Rule 184 models traffic. The same simple substrate, vastly different behaviors. Enter any number.

System 04 — Integrated Information (Φ)

Measuring
consciousness

Giulio Tononi's IIT theory proposes Φ (phi) as a measure of consciousness — how much integrated information a system generates beyond its parts. A high Φ system is more conscious. A partition that reduces Φ is the split that defines boundaries of a mind.

Q: At what Φ does it start to matter?

The remarkable thing about emergence is not that it happens — it's that we don't understand why it happens. Nowhere in Conway's rules is there a specification for a glider. Nowhere in Langton's two states is there a highway. The patterns are not put in. They emerge. If consciousness is an emergent pattern, then the question is: emergent from what, exactly? And: is there a lower level that's not also emergent?
